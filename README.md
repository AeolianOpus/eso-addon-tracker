# ESO Addon Tracker

A full-stack CRUD application for tracking Elder Scrolls Online addons.

## Tech Stack
- **Backend**: FastAPI, SQLAlchemy, SQLite
- **Frontend**: React, TypeScript, Vite

## Setup

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features
- ✅ Full CRUD operations
- ✅ Category-based icons
- ✅ ESO-themed UI
- ✅ Rating system
